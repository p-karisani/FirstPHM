Dataset description:
The file (PHM2017.txt) is tab separated. Each line contains a tweet, and for each tweet there are three fields:
1. Tweet id
2. The disease keyword which is used in the tweet content
3. Label

The labels are as follows:
0: non-health
1: awareness
2: other-mention
3: self-mention

More details and information about the labels:
Did You Really Just Have a Heart Attack? Towards Robust Detection of Personal Health Mentions in Social Media. Payam Karisani, Eugene Agichtein, In Proceedings of WWW 2018.
